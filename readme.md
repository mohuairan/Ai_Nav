# 🚀 AI 助手快速导航 (AI Quick Nav) 使用指南

这是一个浏览器扩展，旨在解决 Gemini 和 ChatGPT 长对话中“回看难”的问题。它会在页面右侧生成一个悬浮目录，点击即可快速跳转到历史提问。

### ✨ 主要功能

* **双平台支持**：自动识别并适配 **Gemini** 和 **ChatGPT**。
* **智能主题 (v7.0)**：支持 **自动跟随(🌗)**、**强制深色(🌙)**、**强制浅色(☀️)** 三种模式，并**自动记忆**你的偏好。
* **一键跳转**：点击侧边栏摘要，平滑滚动到对应的提问位置。
* **完全自由**：支持**拖拽移动**窗口、**缩放大小**、调节**背景透明度**。
* **智能静默**：新消息自动追加，阅读体验极佳。

---

### 📂 第一步：准备文件

1. 在电脑上新建一个文件夹，命名为 `AI_Nav`（或者你喜欢的名字）。
2. 确保文件夹内包含以下 2 个核心文件：
   * `manifest.json`
   * `content.js`
   * *(可选: `styles.css`，如果你的 `content.js` 是 v2.0 及以上版本，样式已内置，可忽略此文件)*

---

### 🛠️ 第二步：安装插件

此插件支持 **Chrome**、**Edge** (Chromium 内核) 以及 **Firefox**。

#### 🟢 Microsoft Edge 用户：

1. 在地址栏输入 `edge://extensions/` 并回车。
2. 在左侧（或右侧）找到 **“开发人员模式”** 开关，将其**打开**。
3. 点击出现的 **“加载解压缩的扩展”** 按钮。
4. 选择你刚才创建的 `AI_Nav` **文件夹**（注意是选文件夹）。

#### 🟡 Google Chrome 用户：

1. 在地址栏输入 `chrome://extensions/` 并回车。
2. 在右上角开启 **“开发者模式”**。
3. 点击左上角的 **“加载已解压的扩展程序”**。
4. 选择 `AI_Nav` **文件夹**。

#### 🟠 Mozilla Firefox 用户：

1. 在地址栏输入 `about:debugging` 并回车。
2. 在左侧菜单点击 **“此 Firefox” (This Firefox)**。
3. 点击中间的 **“临时载入附加组件...” (Load Temporary Add-on)** 按钮。
4. 进入 `AI_Nav` 文件夹，选择 **`manifest.json`** 文件（注意：Firefox 是选文件）。
   * *注意：Firefox 的临时加载模式在关闭浏览器后会失效，下次使用需重新加载。*

> **安装成功后**：回到 Gemini 或 ChatGPT 页面，按下 `F5` 刷新，你应该能看到右侧出现了一个半透明的悬浮窗口。

### 🖱️ 第三步：如何使用

| 功能 | 图标 | 操作方法 |
| :--- | :---: | :--- |
| **切换主题** | 🌗/🌙/☀️ | 点击标题栏的图标，在 **自动** / **深色** / **浅色** 间循环切换。插件会记住你的选择。 |
| **调节透明度** | ➖🔵➖ | 拖动标题栏上的 **滑块**。向左变透明（防遮挡），向右变深色。 |
| **跳转历史** | 📄 | 点击列表中的任意一行，页面会自动平滑滚动到该提问处，并高亮闪烁一下。 |
| **回到最新** | ⬇ | 点击标题栏上的 **向下箭头**，快速滚到底部查看最新回复。 |
| **手动刷新** | ↻ | 点击标题栏上的 **刷新图标**，强制重新抓取列表。 |
| **移动窗口** | ✋ | 按住窗口顶部的 **“标题栏”**（空白处），拖动到屏幕任意位置松开。 |
| **调整大小** | ↘ | 鼠标移动到窗口 **右下角**，按住拖动即可改变宽和高。 |

### ❓ 常见问题 (Q&A)

**Q: "自动模式 (🌗)" 是怎么工作的？**
* **A:** 插件会自动检测网页背景是亮还是暗。如果网页是深色模式，插件自动变黑；如果网页是浅色模式，插件自动变白。

**Q: 插件窗口没有显示怎么办？**
* **A:** 请先刷新网页。如果还没显示，请检查浏览器的扩展管理页面，确认插件已启用且没有报错（如果有报错，点击“错误”查看详情，通常是文件名不对）。

**Q: Firefox 重启后插件不见了？**
* **A:** 这是 Firefox 的安全机制。未签名的开发版插件只能“临时加载”。如果你希望永久安装，需要使用 Firefox 开发者版 (Developer Edition) 并在设置中允许未签名插件，或每次使用前快速加载一次。

**Q: 我更新了代码，需要重装吗？**
* **A:** 不需要。只需保存代码文件 -> 去浏览器扩展页面点击该插件卡片上的 **“刷新”** 或 **“重新加载”** 按钮 -> 再刷新网页即可。
