# 🚀 AI 助手快速导航 (AI Quick Nav) 终极版

这是一个强大的浏览器扩展，专为 **Gemini** 和 **ChatGPT** 用户设计。它不仅解决了长对话“回看难”的问题，还提供了跨对话的**收藏夹**功能，让你可以建立属于自己的常用对话知识库。

### ✨ 核心功能 (v10.1)

* **双平台支持**：自动识别并适配 **Gemini** 和 **ChatGPT**。
* **📑 双模式导航**：
    * **当前大纲**：自动提取当前页面提问，点击一键跳转。
    * **收藏对话**：跨页面管理你的重要对话，随时一键呼出。
* **⭐ 全局收藏夹**：看到有价值的对话，点击顶部“★ 收藏”，永久保存链接。
* **✏️ 自由重命名**：不再依赖系统生成的标题，你可以随意编辑收藏条目的名称（如“周报草稿”、“Python 学习”）。
* **🎨 智能主题**：支持 **自动跟随(🌗)**、**强制深色(🌙)**、**强制浅色(☀️)** 三种模式。
* **🖱️ 极致交互**：
    * **防误触拖拽**：按住标题栏任意空白处即可拖动，操作更丝滑。
    * **自由缩放**：右下角拖拽手柄，随意改变窗口大小。
    * **透明度调节**：顶部滑块调节背景透明度，不遮挡视线。

---

### 📂 第一步：准备文件

1.  在电脑上新建一个文件夹，命名为 `AI_Nav`。
2.  确保文件夹内包含以下 2 个核心文件：
    * `manifest.json`
    * `content.js`
    * *(v10.1 版本已将样式内置于 JS 中，无需 `styles.css` 文件)*

---

### 🛠️ 第二步：安装插件

此插件支持 **Chrome**、**Edge** (Chromium 内核) 以及 **Firefox**。

#### 🟢 Microsoft Edge 用户：
1.  地址栏输入 `edge://extensions/` 回车。
2.  开启左侧/右侧的 **“开发人员模式”**。
3.  点击 **“加载解压缩的扩展”**。
4.  选择你创建的 `AI_Nav` **文件夹**。

#### 🟡 Google Chrome 用户：
1.  地址栏输入 `chrome://extensions/` 回车。
2.  开启右上角 **“开发者模式”**。
3.  点击左上角 **“加载已解压的扩展程序”**。
4.  选择 `AI_Nav` **文件夹**。

#### 🟠 Mozilla Firefox 用户：
1.  地址栏输入 `about:debugging` 回车。
2.  点击左侧 **“此 Firefox” (This Firefox)**。
3.  点击 **“临时载入附加组件...”**。
4.  进入 `AI_Nav` 文件夹，选择 **`manifest.json`** 文件。
    * *注意：Firefox 临时加载模式在重启浏览器后需重新加载。*

### 🎮 第三步：使用指南

插件界面分为顶部控制栏和两个标签页（Tabs）。

#### 1. 顶部控制栏 (Header)

| 图标/按钮 | 功能描述 |
| :--- | :--- |
| **★ 收藏** | **收藏当前对话**。点击后，当前网页会被存入“收藏对话”标签页中。 |
| **🌗/🌙/☀️** | **切换主题**。在自动/深色/浅色模式间循环切换。插件会记忆你的选择。 |
| **↻** | **刷新列表**。如果页面内容更新了但列表没变，点它强制刷新。 |
| **标题栏** | **拖拽区域**。按住“Gemini 助手”文字或空白处，可拖动窗口位置。 |

#### 2. 标签页：当前大纲 (Current Outline)
* **功能**：显示当前正在进行的对话中的所有提问。
* **操作**：点击列表项，网页会自动平滑滚动到对应的提问位置，并高亮闪烁提示。

#### 3. 标签页：收藏对话 (Saved Chats)
这是你的私人知识库，功能如下：

* **跳转**：点击条目，直接打开对应的对话链接。
* **✏️ 重命名 (新功能)**：
    1.  将鼠标悬停在某个收藏条目上。
    2.  点击出现的 **✎ (笔图标)**。
    3.  输入新名字，按 **回车 (Enter)** 或点击空白处保存。
* **✕ 删除**：
    1.  将鼠标悬停在条目上。
    2.  点击 **✕ (叉图标)** 即可从收藏夹移除。

### ❓ 常见问题 (Q&A)

**Q: 我的收藏数据保存在哪里？**
* **A:** 数据保存在浏览器的 `localStorage` 中。只要你不清除浏览器缓存（Local Storage 数据），收藏就会一直存在。

**Q: 为什么拖动窗口时有时候拖不动？**
* **A:** (v10.1 已优化) 请按住标题文字（如 "ChatGPT 助手"）或标题栏的空白背景进行拖动。为了防止误触，按住按钮或输入框时无法拖动窗口。

**Q: 为什么收藏时名字是“未命名对话”？**
* **A:** 插件默认会尝试抓取网页标题。如果网页还没加载完或者标题是通用的（如 "New Chat"），它会显示默认名。您可以点击 **✎** 按钮手动把它改成你喜欢的名字。

**Q: Firefox 重启后插件消失了？**
* **A:** 这是 Firefox 的安全机制。建议使用 Chrome/Edge 获得永久安装体验，或者在 Firefox 开发者版中开启未签名扩展支持。
