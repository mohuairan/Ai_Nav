# 🚀 AI 助手快速导航 (AI Quick Nav) 全平台版

![AI Quick Nav 宣传海报](image\poster.png)

这是一个强大的浏览器扩展，专为 **Gemini**、**ChatGPT** 和 **DeepSeek** 用户设计。它不仅解决了长对话“回看难”的问题，还提供了跨对话的**收藏夹**功能，让你可以建立属于自己的常用对话知识库。

### ✨ 核心功能 (v12.2 完美交互版)

* **全平台支持**：完美适配 **Gemini**、**ChatGPT** 以及 **DeepSeek (chat.deepseek.com)**。
* **🤖 悬浮球模式**：支持**最小化为悬浮球**，小巧不占地，点击即可快速展开/收起。
* **📑 双模式导航**：
    * **当前大纲**：自动提取当前页面提问，点击一键跳转。
    * **收藏对话**：跨页面管理你的重要对话，随时一键呼出。
* **⭐ 全局收藏夹**：看到有价值的对话，点击顶部“★ 收藏”，永久保存链接。
* **✏️ 自由重命名**：收藏的对话支持手动重命名。
* **🎨 智能主题**：支持 **自动跟随(🌗)**、**强制深色(🌙)**、**强制浅色(☀️)**。
* **🖱️ 极致交互体验 (New!)**：
    * **丝滑动画**：最小化和展开时拥有流畅的过渡动画。
    * **零延迟缩放**：采用“按需动画”技术，手动调整窗口大小时绝对跟手，毫无延迟。
    * **智能归位**：从小球恢复大窗口时，如果窗口超出屏幕边缘，会自动回弹到可视区域。
    * **防误触逻辑**：精准识别“点击”与“拖拽”，拒绝拖动悬浮球后意外展开窗口。
    * **尺寸记忆**：实时记住你调整过的窗口大小，下次展开完美还原。

### 📂 第一步：准备文件

1.  在电脑上新建一个文件夹，命名为 `AI_Nav`。
2.  确保文件夹内包含以下 2 个核心文件：
    * `manifest.json`
    * `content.js`
    * *(样式已内置于 JS 中，无需 `styles.css` 文件)*

---

### 🛠️ 第二步：安装插件

此插件支持 **Chrome**、**Edge** (Chromium 内核) 以及 **Firefox**。

#### 🟢 Microsoft Edge 用户：
1.  地址栏输入 `edge://extensions/` 回车。
2.  开启左侧/右侧的 **“开发人员模式”**。
3.  点击 **“加载解压缩的扩展”**。
4.  选择你创建的 `AI_Nav` **文件夹**。

#### 🟡 Google Chrome 用户：
1.  地址栏输入 `chrome://extensions/` 回车。
2.  开启右上角 **“开发者模式”**。
3.  点击左上角 **“加载已解压的扩展程序”**。
4.  选择 `AI_Nav` **文件夹**。

#### 🟠 Mozilla Firefox 用户：
1.  地址栏输入 `about:debugging` 回车。
2.  点击左侧 **“此 Firefox” (This Firefox)**。
3.  点击 **“临时载入附加组件...”**。
4.  进入 `AI_Nav` 文件夹，选择 **`manifest.json`** 文件。
    * *注意：Firefox 临时加载模式在重启浏览器后需重新加载。*

### 🎮 第三步：使用指南

#### 1. 顶部控制栏 (Header)

| 图标/按钮 | 功能描述 |
| :--- | :--- |
| **★ 收藏** | **收藏当前对话**。点击后，当前网页会被存入“收藏对话”标签页中。 |
| **🌗/🌙/☀️** | **切换主题**。在自动/深色/浅色模式间循环切换。插件会记忆你的选择。 |
| **－** | **最小化**。点击此按钮（或**双击标题栏**），窗口会收起变成一个悬浮球。 |
| **标题栏** | **拖拽区域**。按住文字或空白处可拖动窗口。 |

#### 2. 🤖 悬浮球模式 (Floating Ball)
当你觉得窗口挡视线时，可以把它变小：
* **如何收起**：点击标题栏右上角的 **“－”**，或直接 **双击标题栏**。
* **如何展开**：点击屏幕上的 **“🤖”** 悬浮球图标。
* **如何移动**：按住悬浮球任意位置，即可随意拖拽到屏幕角落。

#### 3. 标签页功能
* **当前大纲**：显示当前对话目录，点击跳转。
* **收藏对话**：
    * **跳转**：点击条目直接打开。
    * **✏️ 重命名**：悬浮条目 -> 点击 **✎** -> 输入新名 -> 回车保存。
    * **✕ 删除**：悬浮条目 -> 点击 **✕** 移除。

### ❓ 常见问题 (Q&A)

**Q: DeepSeek 页面不显示插件怎么办？**
* **A:** 请检查 `manifest.json` 文件中是否包含 `chat.deepseek.com`。修改 manifest 后需要去扩展管理页面点击“刷新”按钮。

**Q: 为什么拖动窗口时有时候拖不动？**
* **A:** 为了防止误触，按住按钮或输入框时无法拖动。请按住标题文字或空白背景进行拖动。

**Q: 窗口大小调整后，下次打开会变回原样吗？**
* **A:** 不会。v12.0+ 版本引入了 `ResizeObserver`，它会实时记住你调整过的大小，下次展开时会自动还原。

**Q: 我的收藏数据保存在哪里？**
* **A:** 数据保存在浏览器的 `localStorage` 中。只要你不清除浏览器缓存（Local Storage 数据），收藏就会一直存在。

**Q: DeepSeek 页面不显示插件怎么办？**
* **A:** 请检查 `manifest.json` 文件中是否包含 `chat.deepseek.com`。修改 manifest 后需要去扩展管理页面点击“刷新”按钮。

**Q: 悬浮球拖动后，展开时窗口位置乱了怎么办？**
* **A:** (v11.2+ 已修复) 现在的版本已经完美解决了这个问题。无论你把球拖到哪里，展开时窗口都会稳稳地停留在那里，且保持原有的大小。
